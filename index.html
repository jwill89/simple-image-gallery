Tag Help

Tag Categories

Tags have five categories they belong to, each having a different color when added they appear in an item's tag list. When you add a tag, you can add a shortcode followed by a colon before the tag to ensure it is added to the appropriate category. The shortcodes are as follows:

Tag Shortcode Description
General g: General terms that describe features of the image or video. This is the default if no shortcode is used.
Artist a: The artist of the work. This is not the artist of the original work the characters might belong to, but the name or alias of the person who created this specific piece.
Character c: The name of a character in the image or video. We use a "<First Name> <Last Name>" format for characters with a first and last name. For example, "Kurosaki Ichigo" would be c:Ichigo Kurosaki. If the character is known by a single name, such as "Goku", then it would be c:Goku.
Source s: The source material of the characters in the piece. This could be a movie, video game, anime, manga, manhwa, etc. For works in foreign languages with non-roman characters, please use the romanization of the title.
Personal List p: These are personal lists that may be created for individuals to record their favorite pieces. We dissuade the use of these in favor of saving your favorite list of tags.
